Quick SNLI Predict Model 
===
Implements MLP based model which achieve the same effect as lstm/gru with tensorflow low api

## Resources  
- [Datasets](https://nlp.stanford.edu/projects/snli)
- [Code base on](https://github.com/Smerity/keras_snli)
- [glove.840B.300d.txt](http://nlp.stanford.edu/data/glove.840B.300d.zip)

## Implments with Tensorflow low api
- Support Multi GPU
- TF datasets API. Hanlde datas which much more than memory.
- BN\L2\Dropout to prevent overfit
- Early stop
- Log with tqdm and tensorboard

## run
- parse data to format: 'label sentence1 sentence2'
- build tf-record
```
python prepro.py
``` 
- modes
```
python snli.py
```
